Multi-car Parking System

GitHub

Languages: Java

Tools: JavaFX, JSON

This project is an individual assignment that I have completed in my first year. The aim of the assignment was to create a car parking system for different vehicle classes. The brief only asked for a command-line application however, I took the initiative to implement a graphical representation of the application using the JavaFX library.

For the application, I used a Model-View-Controller (MVC) architecture approach to organise the structure. The idea behind this is so that each parts that makes the application. This allows the overall functionality of the application to stay intact when one part of the application changes.

The main features of the application includes:

  • Login as two types of users, admins (the right to manage garage attendants) and customers
  • Register a ticket for different types of vehicle
  • Requesting for parking assistance and retrieval of vehicle where a garage attendant will be assigned to if available
  • Ability to calculate ticket pricing for different class of vehicle
  • Allows the customer to pay and exit the carpark
  • Shows the current parking condition of the carpark
  • Allows admins to manage garage attendants
