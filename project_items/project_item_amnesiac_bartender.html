Amnesiac Bartender

GitHub

Languages: C#

Tools: Unity

Amnesiac Bartender is a puzzle and arcade game that I submitted for the Ludum Dare 42 game jam. I worked on this game with one other member. My primary role was the coder, coding the mechanics of the games while the other member is responsible for art and design. By applying what we learnt in our university, the overall workflow went smoothly and well-organised.

The main goal of this game is to create bar drinks using the ingredients on the table. There is a recipe book which tells the player which ingredients to mix to create the appropriate drinks for the customer. The customers will tell the player which drink they would like to order and based on the description given, find the correct drink recipe and mix it. Also, the rating bar indicates how successful you are managing your bar in the span of 7 days.

For more details or if you would like to play the game, visit the ludum dare or itch.io page.
