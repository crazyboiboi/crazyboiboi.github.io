Cube Shooter

GitHub

Languages: C#

Tools: Unity

Cube Shooter is endless 3D shooter game with different types of enemy that chases the player down. The main objective of the game is to survive and score points by eliminating the the enemies. The project started of as a stepping stone for my journey in game development.

Some of the main features of the game includes:

  • Main menu and options menu
  • Health and stamina system
  • 3 types of enemies => Tanky (pink), Speedy (green), Normal (orange)
