Light My Way

Languages: C#

Tools: Unity

Light My Way is 2D platforming game with various challenges and levels. It uses basic geometric shapes as it is partly inspired by a game called "Thomas was Alone". The main objective of the game is to collect Light objects (keys) to unlock and proceed to the next level.

Some of the main features of the game includes:

  • Player movement including the ability to double jump
  • Player interaction with objects in the scene
  • Different types of platforms (moving, activatable, one-sided)
  • Experimentation with simulating Water properties
