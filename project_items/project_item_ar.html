Augmented Reality for Heritage

Languages: C#

Tools: Unity, Vuforia AR SDK, ColMap, MeshLab

This project is a third year dissertation that I have completed during my undergraduate study at Aberystwyth University. It is about implementing object-based registration as an additional feature for an existing AR application created for "Experience the Lost and Invisible" project which targets a prehistorical site at Bryn Celli Ddu Neolithic passage tomb located in Anglesey, UK.

The final application is successful in achieving object-based recognition for AR. However, due to the Covid-19 pandemic, I was not able to carry out testing on the actual site thus an alternative would be to use the rock sculpture located at the front of the university's Computer Science building for testing purposes. At the moment, the application is able to track two types of objects, mug and the rock sculpture object. It is possible to include more objects through the provision of 3D model of the object to be recognised.

For the project, it also uses the photogrammetry technique to capture and reconstruct the 3D model of a real life object such as the rock sculpture. It involves taking lots of images of the object then processing it using ColMap and MeshLab to obtain the final 3D model of the object.

There are two types of recognition mode available:

  • Standard - uses guide views to inform the user how to position the camera to achieve recognition
  • Advanced - model data is trained to allow recognition from multiple angle
